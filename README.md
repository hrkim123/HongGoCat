# HongGoCat 🐱

Steam의 Bongo Cat처럼 **키보드/마우스 입력에 맞춰 고양이가 연주하는 전체화면 데스크톱 오버레이**입니다. 화면 전체를 덮는 투명·클릭통과 창이라 평소엔 다른 앱을 그대로 쓰면서, 고양이 위젯 위에서만 상호작용합니다. 친구들과 방을 만들어 함께 두드리고, `Ctrl+Shift+M`으로 마우스를 따라가는 미사일을 쏴 서로를 맞힐 수 있습니다.

> 단축키: `Ctrl+Shift+B` 채팅 · `Ctrl+Shift+M` 미사일 발사(커서 유도)

## 실행 방법

```bash
npm install

# 1) 릴레이 서버 (방장 한 명 또는 공용 서버에서)
npm run server        # ws://<호스트IP>:8787

# 2) 오버레이 앱 (참가자 각자)
npm start
```

오버레이는 책상 위에 **카운터(누적 상호작용 횟수)**와 **⚙ 설정 버튼**만 컴팩트하게 얹혀 있습니다. ⚙를 누르면 **별도의 설정 창**이 열립니다(오버레이 창 크기는 그대로). 거기서 이름·털색·무늬·모자·편집 모드·서버/방을 조정합니다.

- 오버레이는 투명·프레임리스(타이틀바 없음)이며, 캐릭터 부분을 드래그해 이동합니다.
- 접속하지 않아도 혼자 모드로 동작합니다.

## 꾸미기 🎨

설정 창에서 실시간으로 바꿀 수 있고, 방에 접속 중이면 친구들 화면에도 즉시 반영됩니다.

- **털색** 9종: 흰색/크림/회색/갈색/검정/치즈/분홍/민트/라벤더
- **무늬** 5종: 단색 / 태비(줄무늬) / 턱시도 / 점박이 / 포인트
- **모자** 6종: 없음/비니/파티모자/왕관/중산모/야구모자

### 디자인 편집 모드 ✏️

설정 창의 "디자인 편집 모드"를 켜면 오버레이에 **귀·눈·꼬리 핸들**이 나타납니다. 드래그해서 위치를 옮기면 즉시 반영되고 저장됩니다(localStorage `feat`). 나만의 고양이 표정/실루엣을 만들 수 있어요.

새 무늬·색을 추가하려면 [renderer/animals.js](renderer/animals.js)의 `PATTERNS`/`patternHead`/`patternBody`, `SKINS`만 손보면 됩니다.

## 함께 하기 (멀티플레이)

한 명이 릴레이 서버를 켜고, 나머지는 그 서버 주소로 접속합니다.

**같은 Wi-Fi/LAN** — 서버를 켠 컴퓨터의 사설 IP로 접속:
```bash
npm run server         # 서버 PC에서 실행
# 다른 사람은 앱에서 서버 주소를 ws://<서버PC_IP>:8787 로 입력
```
현재 이 PC의 IP는 `172.30.16.175` 입니다 → 같은 네트워크 친구는 `ws://172.30.16.175:8787` 입력.
(윈도우 방화벽에서 8787 포트 인바운드 허용이 한 번 필요할 수 있어요.)

**인터넷으로 (다른 장소의 친구와)** — 셋 중 하나:
- **터널** (가장 쉬움): 서버 PC에서 `npx localtunnel --port 8787` 또는 `cloudflared tunnel --url http://localhost:8787` 실행 → 나오는 주소를 `wss://...` 로 접속
- **클라우드 배포**: `server/` 를 Fly.io / Railway 등에 올리고 `wss://<배포주소>` 로 접속 (무료 티어로 충분)
- **포트포워딩**: 공유기에서 8787 포트를 서버 PC로 열고 공인 IP로 접속 (보안상 비권장)

> 방 코드는 아무 문자열이나 정하면 됩니다 (예: `BONGO123`). 같은 코드끼리 한 방에 모여요. 방당 최대 12명.

## 채팅 💬

오버레이는 평소 포커스가 없으므로, 채팅은 **명시적으로 열어서** 입력합니다:

1. 어디서든 **`Ctrl+Shift+B`** (또는 오버레이의 💬 버튼) → 하단에 입력창이 열림
2. 메시지 입력 후 **Enter** → 전송되고 입력창이 닫히며 원래 작업으로 포커스 복귀 (**Esc** = 취소)
3. 메시지는 캐릭터 머리 위 말풍선으로 몇 초간 표시됨 (최대 80자, 서버에서 0.4초 스팸 가드)

글로벌 훅으로 타이핑 내용을 수집하는 방식이 아니라, 포커스된 입력창에 직접 타이핑하는 방식이라 프라이버시 원칙이 유지됩니다.

## 구조

```
main.js            Electron 메인 — 투명 오버레이 창 + 별도 설정 창 + uiohook 글로벌 훅
preload.js         두 창(오버레이/설정)용 contextBridge API + IPC 중계
renderer/
  index.html/app.js/style.css      오버레이 — 렌더 루프·입력·WebSocket·편집 모드
  settings.html/settings.js/.css   별도 설정 창 (IPC로 오버레이 제어)
  animals.js                       고양이 벡터 드로잉 (무늬·모자·편집 오프셋) — 스프라이트로 교체 가능
server/server.js   방 코드 기반 릴레이 서버 (ws, 방당 최대 12명)
```

## 프라이버시 원칙

네트워크로는 `{ kind: 'key' | 'mouse' }` **펄스 신호만** 전송합니다.
어떤 키를 눌렀는지는 절대 수집·전송하지 않습니다 (초당 25펄스로 제한).
