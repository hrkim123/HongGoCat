ⓘ 무기 설명
🚀 미사일

커서를 따라가 명중 시 폭발(범위 피해). 여러 발이 겹치면 합쳐져 커지고(최대 5) 황금색이 됩니다. 왼쪽 클릭으로 마지막 방향 3배 속도 부스터. (기본 무료)

🛡 쉴드

커서 방향에 방패를 10초간 소환해 미사일을 막습니다. 체력 10 — 맞을수록 금이 가다 깨지고 3초 쿨타임.

🔫 게틀링건

커서에 터렛 거치 → 왼쪽 클릭 누르는 동안 연사. ~2초 사격 시 과열(3초, 연기). 체력 10, 총알당 데미지 0.3. 인간과 동시 소환 불가.

🐜 개미

커서 위치에 소환(최대 5). 작업 표시줄 위를 기어다니며 상대 개미·터렛과 싸움. 체력 1.

🕺 인간

커서에 소환, W/A/S/D로 이동(점프·빠른낙하). 체력 5, 상대 무기와 충돌. 게틀링과 동시 소환 불가. (본인에게만 보임)

🕳 블랙홀

커서에 10초 소환(쿨 60초). 범위 안 캐릭터 외 모든 무기를 중심으로 빨아들여 소멸. 상점 전용.

단축키: Ctrl+Alt+1 / 2 / 3 으로 각 슬롯 무기 사용
